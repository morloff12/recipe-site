In a large mixing bowl, combine: 220 g cold milk 180 g cold whipping cream 100–120 g cold espresso Blend for a few seconds. ✅ Best Practice for AnyList Compatibility

AnyList appears to prefer Microdata embedded directly in HTML, structured exactly like what it scrapes from recipe blogs. While JSON-LD works for some tools (like Google), AnyList is optimized for Microdata, and expects:

  • itemprop="recipeIngredient" on each <li>

  • itemprop="recipeInstructions" on each <li> or <div>, with no nesting blocks

  • A separate <meta itemprop="image"> tag OR an <img> tag with itemprop="image"

  • Simple <ul> and <ol> structures (no extra <div> wrappers)

So yes, we can and should do this with a standalone HTML file using Microdata, just ensure the format is blog-style, like this:
